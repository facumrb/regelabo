# Información de Directorio: doc/

## Propósito exhaustivo
- **`Model_Overview.png`**: Diagrama de bloques visual de muy alto nivel para el usuario con el esquema esquemático general, para asimilar la estructura biológica (basilar membrane, hair cell) a cómo se refleja en el código informático.
- **Archivos de Notas (.pdf)**: Las liberaciones como `Verhulstetal2018-v1.2-extended-release-note.pdf` fungen como material del desarrollador, apuntando revisiones matemáticas menores o parches que se incluyeron a la simulación posteriormente.
- **`Verhulstetal2018.pdf`**: El artículo original o investigación (paper científico), donde se formula matemáticamente toda la anatomía del proyecto.

## Flujo de ejecución
Al tratarse de material estático de estudio y lectura para el ser humano, no cuenta de flujo de ejecución.

## Elemento más relevante
**Archivo `Verhulstetal2018.pdf`**: Es la piedra angular bibliográfica de donde se deducen cada una de las integrales, las variables de acople y los supuestos neuronales que posibilitan el modelo. Si se desean aplicar cambios a la respuesta de onda o mejorar las no linealidades, la consulta obligatoria es a este documento científico.
