# workflows/  
Contiene los slash commands de SDD (/speckit.*).
Cada archivo .md representa un paso del ciclo de desarrollo:

    speckit.specify.md → definir requerimientos.

    speckit.clarify.md → aclarar dudas.

    speckit.plan.md → planificar.

    speckit.tasks.md → dividir en tareas.

    speckit.analyze.md → analizar.

    speckit.implement.md → implementar.

    speckit.checklist.md → checklist de validación.

    speckit.constitution.md → reglas base del proyecto.
