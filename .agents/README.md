# Agentes Globales
Esta carpeta contiene la configuración global de los agentes:
- `workflows/`: Archivos markdown con los prompts de cada comando (ej. `/speckit.plan`).
- `skills/`: Habilidades globales que el agente puede usar (ej. herramientas de análisis de repositorio, conexión a la DB).
- `rules/`: Reglas de inferencia base para el agente.
